# weather-dashboard 

Weather Analytics Dashboard | Power BI

1.  Project Title / Headline
    An interactive and visually engaging dashboard built using Microsoft Power BI Desktop to analyze real-time and forecast weather data. The dashboard helps users understand temperature trends, air quality, and weather conditions for better planning and decision-making.
    
3. Short Description / Purpose
   The dashboard was built using the following tools and technologies:
  
📊 Power BI Desktop – Data visualization and dashboard creation
🔄 Power Query – Data cleaning and transformation
🧮 DAX (Data Analysis Expressions) – Creating calculated measures and KPIs
📁 Excel / CSV Dataset – Data source
🔗 Data Modeling – Relationships between tables

3. Data Source
Weather dataset containing:
Temperature
Humidity
Wind Speed
Air Quality Index (AQI)
Chance of Rain
Data includes daily and weekly forecast information

4. Features / Highlights
🌡️ Current Weather Overview (Temperature, condition, location)
📈 Forecast Analysis (Line chart showing temperature trends)
🌧️ Chance of Rain Visualization (Bar chart)
🌫️ Air Quality Index (AQI) (Donut chart with pollution levels)
🌅 Sunrise & Sunset Timing
🌬️ Additional Metrics (Humidity, Wind Speed, Pressure, Visibility, UV Index)

5. Key Insights
Identified temperature trends across multiple days
Compared rain probability across the week
Analyzed air quality levels (PM10, NO2, etc.)
Helped in understanding weather patterns for planning activities.

6. Outcome / Learning
Improved skills in:
Data cleaning
Data visualization
Dashboard design
Gained hands-on experience in Power BI and DAX
Learned how to present data in a user-friendly and interactive way
